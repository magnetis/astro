# Buttons

## Primary

Used for main action in a section or screen. Shouldn't be used with other primary together.

<Playground>
  <button className="btn btn--primary">
    Primary
  </button>
  
  <button className="btn btn--primary" disabled>
    Primary disabled
  </button>
</Playground>

## Secondary

Secondary buttons are used when presenting secondary actions when with a primary action.

<Playground>
  <button className="btn btn--secondary">
    Secondary
  </button>
 
  <button className="btn btn--secondary" disabled>
    Secondary disabled
  </button>
</Playground>
