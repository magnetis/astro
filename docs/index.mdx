### install

* Add Astro as a dependency:

```json
  “dependencies”: {
    “@magnetis/astro”: “git+ssh://github.com/magnetis/astro.git”
  }
```

* Get our font links in the [Typography page](https://magnetis.github.io/astro/#/docs-typography) and load them in your project.

[Read install docs](https://github.com/magnetis/astro/blob/master/README.md#adding-astro-to-your-project)

### usage

* Astro components are atomic and can be used in isolation.

```jsx
  import '../src/index.css';

  <button className="btn btn--uranus btn--medium">
    primary
  </button>
```

[Read usage docs](https://github.com/magnetis/astro/blob/master/README.md#using-astro)
