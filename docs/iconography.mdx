## icons usage

### minimum

Pick one icon name above and apply the following class structure:

<div className='code-snippet'>
  <pre>{'<i className="a-icon a-icon__icon-name" />'}</pre>
</div>

### sizes

You may apply the following CSS modifiers to get smaller or bigger icon sizes.

<Playground>
  <i className='a-icon a-icon__cake a-icon--size-small' />
  <i className='a-icon a-icon__cake a-icon--size-medium' />
  <i className='a-icon a-icon__cake a-icon--size-large' />
</Playground>

### spacers

You may apply the following CSS modifiers to apply left or right spacers.

<Playground>
  <div>
    <i className='a-icon a-icon__profile a-icon--space-right' />
    <span className='a-text--medium'>Astro Team</span>
  </div>
  <div>
    <span className='a-text--medium'>Astro Team</span>
    <i className='a-icon a-icon__profile a-icon--space-left' />
  </div>
</Playground>
