# buttons

## primary

Primary buttons are medium sized and have a colored background. They aren't always blue, though, so there isn't a "primary" CSS class - primary here is more of a concept.
Please note that the color and size are explicitly applied to the `<button>` element.
Use these for the main action in a section or screen. You should display only one primary button at a time.

<Playground>
  <button className="btn btn--uranus btn--medium">
    primary
  </button>
  
  <button className="btn btn--uranus btn--medium" disabled>
    primary disabled
  </button>
</Playground>

## colors

Buttons can be used in different colors just by changing their CSS color class.

<Playground>
  <button className="btn btn--uranus btn--medium">
    uranus
  </button>
 
  <button className="btn btn--venus btn--medium">
    venus
  </button>

  <button className="btn btn--earth btn--medium">
    earth
  </button>
</Playground>

## sizes

Buttons can be used in different sizes, also by changing their CSS size class.

<Playground>
  <button className="btn btn--uranus btn--small">
    small
  </button>
 
  <button className="btn btn--uranus btn--medium">
    medium
  </button>

  <button className="btn btn--uranus btn--large">
    large
  </button>
</Playground>

## outline

Outline buttons can be used as secondary. Secondary buttons present secondary actions when there's already a primary action in the same section or screen.
Please note that there are color variations to the outline CSS classes. You also need to specify their size with one of the size CSS classes.

<Playground>
  <button className="btn btn--outline-uranus btn--small">
    outline uranus
  </button>
  <button className="btn btn--outline-uranus btn--medium">
    outline uranus
  </button>
  <button className="btn btn--outline-uranus btn--large">
    outline uranus
  </button>
 
  <button className="btn btn--outline-uranus btn--medium" disabled>
