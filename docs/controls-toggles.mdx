# controls & toggles

## radio buttons

Radio buttons appear when the user must select only one option from more than two.

Mouseover to view `:hover` state.

</Playground>

## checkboxes

Checkboxes are used when a user might select multiple options from a list, or when a single specific action is required - like reading a contract.

Mouseover to view `:hover` state.
