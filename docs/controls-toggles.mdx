# controls & toggles

## radio buttons

Radio buttons appear when the user must select only one option from more than two.

Mouseover to view `:hover` state.

<Playground>
  <div className="a-radio">
    <input type="radio" id="radio1" name="radio" />
    <label htmlFor="radio1">Radio</label>
  </div>

  <div className="a-radio">
    <input type="radio" id="radio2" name="radio" checked />
    <label htmlFor="radio2">Selected</label>
  </div>

  <div className="a-radio a-radio--disabled">
    <input type="radio" id="radio3" name="radio" disabled />
    <label htmlFor="radio3">Disabled</label>
  </div>
</Playground>

## checkboxes

Checkboxes are used when a user might select multiple options from a list, or when a single specific action is required - like reading a contract.

Mouseover to view `:hover` state.
