# controls & toggles

## radio buttons

Radio buttons appear when the user must select only one option from more than two.

Please follow the markup structure from the examples below. You must create a unique `id` for each radio button shown in the same page.

Mouseover to view `:hover` state.

<Playground>
  <div className="a-radio">
    <input type="radio" id="radio1" name="radiogroup1" />
    <label htmlFor="radio1">Radio</label>
  </div>

  <div className="a-radio">
    <input type="radio" id="radio2" name="radiogroup1" checked />
    <label htmlFor="radio2">Selected</label>
  </div>

  <div className="a-radio a-radio--disabled">
    <input type="radio" id="radio3" name="radiogroup1" disabled />
    <label htmlFor="radio3">Disabled</label>
  </div>
</Playground>

Default label behavior is top alignment. You can add the `a-radio--align-middle` modifier class to switch to middle alignment.

## sliders

Sliders give users a visual indication of adjustable content, mostly numerical (for time, money, etc). They can be used with or without text labels.

Adjust `min` and `max` attributes to set the value range. If you need to manipulate the slider handle position, use the `value` attribute. You must also set an unique `id` for each slider in your page.

We recommend you set the slider width in the `a-slider` class.

The colored percentage fill and counter label will need JS to work on your project. You may find the code for this working example in our GitHub repository. You can also wrap the slider with a React component or apply other frameworks.

Mouseover and click to view `:hover` and `:active` state.

<Playground>
  <div className="a-slider">
    <input type="range" min="1" max="100" id="slider1" />
    <label htmlFor="slider1">
      Slider
      <span className="a-slider--value">50 coins</span>
    </label>
  </div>

  <div className="a-slider">
    <input type="range" min="1" max="100" id="slider1" disabled />
    <label htmlFor="slider1">
      Disabled
      <span className="a-slider--value">0 coins</span>
    </label>
  </div>
</Playground>
